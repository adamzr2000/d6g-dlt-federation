# Create local geth network configuration
```bash
./create_config.sh
```

# Start geth network
```bash
./start_geth_net.sh --file local-geth-network.yml
```

# Stop geth network
```bash
sudo ./stop_geth_net.sh --file local-geth-network.yml
```

# Delete local geth network configuration
```bash
./delete_config.sh
```
